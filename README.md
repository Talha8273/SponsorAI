#  SponsorAI: Kurumsal Sponsorluk CRM Yönetim Sistemi

SponsorAI, üniversite kulüpleri, öğrenci toplulukları ve organizasyonlar için geliştirilmiş, bulut tabanlı ve gerçek zamanlı bir **Müşteri İlişkileri Yönetimi (CRM)** uygulamasıdır. 
Yüzlerce firma ile yürütülen sponsorluk ve iş birliği süreçlerini tek bir merkezden, hatasız ve hızlı bir şekilde yönetmek için tasarlanmıştır.

##  Öne Çıkan Özellikler

* ** Gerçek Zamanlı Bulut Veritabanı (Firebase):** Tüm firma verileri, görüşme notları ve bütçe planlamaları anlık olarak Firestore üzerinde tutulur. Veri kaybı riski sıfırdır.
* ** Toplu Mail (BCC) Motoru:** Seçilen firmaların kurumsal e-posta adreslerini otomatik tarar ve tek tıkla gizli (BCC) formatında toplu mail gönderim ekranını (Outlook/Gmail) açar.
* ** Akıllı Mail Şablonları:** `[FİRMA_ADI]`, `[SEKTÖR]` gibi dinamik değişkenleri kullanarak her firmaya saniyeler içinde özel, kişiselleştirilmiş profesyonel teklif metinleri oluşturur.
* ** Sektör ve Durum Filtreleme:** Gıda, Kozmetik, Kırtasiye vb. sektörlere göre hızlı filtreleme; "Beklemede", "Görüşülecek", "Reddedildi" gibi aşama takipleri.
* ** Modern ve Responsive Arayüz:** Tailwind CSS ile tasarlanmış, kullanıcı dostu ve akıcı bir deneyim (Dark/Light mod desteği eklenebilir altyapı).

##  Kullanılan Teknolojiler

* **Frontend:** React (Vite), TypeScript, Tailwind CSS
* **Backend / Database:** Firebase (Firestore)
* **İkonlar:** Lucide React

##  Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Repoyu bilgisayarınıza klonlayın:
   `git clone https://github.com/Talha8273/SponsorAI.git`
2. Proje dizinine girin:
   `cd SponsorAI`
3. Gerekli paketleri yükleyin:
   `npm install`
4. Geliştirici sunucusunu başlatın:
   `npm run dev`
